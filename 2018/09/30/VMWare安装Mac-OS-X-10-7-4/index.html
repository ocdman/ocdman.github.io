VMWare安装Mac OS X 10.7.4

September 30 2018

之前说过要研究iOS插件开发，那么首先需要安装开发环境。受限于越狱设备的系统只有iOS 6.0，我只有安装XCode 4.5到XCode 5.0.2之间的版本才能进行iOS 6插件开发。而支持这个开发环境的系统只能在Mac OS X 10.7.4到Mac OS X 10.8.4之间。具体对应关系，可以参考XCode维基百科。

由于我的Mac的系统是10.10.4，最低只能安装XCode 7，那么只能曲线救国了。通过虚拟机安装一个Mac OS X 10.7.4系统以及在这个系统下安装XCode 4的最后一个版本XCode 4.6.3。虚拟机的话，我就不在Mac上安装了，因为比较吃性能，索性找一台Windows台式机来安装。

首先我们需要一个VMWare虚拟机。我选择的是VMWare Workstation 12 Pro，版本是v12.1.1，可以从这里下载。

随便找了一个VMware 12专业版的永久许可证的密钥，5A02H-AU243-TZJ49-GTC7K-3C61N，亲测可用。安装过程略过。

安装VMWare后，由于VMWare本身不支持Mac OS X的安装，需要通过打补丁的方式才能安装Mac OS X。可以去github下载unlocker，因为我是Windows系统，右键单击 win-install.cmd，选择以管理员身份运行。等待安装完毕即可。

接下来就是下载Mac OS X 10.7.4镜像文件，由于网上只有DMG文件，需要将其转换成ISO文件，可以在这个网址下载种子。下载完毕后，通过UltraISO这款软件进行转换。

打开VMWare创建一个虚拟机，导入刚才转换好的ISO文件。注意，创建好之后不要急着打开，在虚拟机的安装目录里，编辑.vmx文件，在smc.present="TRUE"后面添加一行
