WD Elements移动硬盘读写速度测试

June 26 2018

这两天一直在研究磁盘的读写速度，以我的Macbook Pro为例，得益于Apple SSD，磁盘的读写速度达到了

Read Speed is: 1151.3 MB/s
Write Speed is: 1223.73 MB/s

于是想到了我手头还有一块WD的移动硬盘，但每次写入文件的时候都感觉不快，顺便也测试一下。

测试命令还是之前在Macbook测试磁盘读写速度一文里介绍的，只不过需要cd到你的移动硬盘目录下去执行命令。测试之后的结果如下

Read Speed is: 41.589 MB/s
Write Speed is: 28.8925 MB/s

和mbp一对比就发现了，读写速度完全不在一个级别上。但是为什么会差这么多呢？好歹我的移动硬盘支持USB 3.0协议啊。我的mbp的USB接口类型是3.0，WD移动硬盘的接口类型也是3.0，而且USB连接线确认过是3.0接口无误，那么理论上移动硬盘的读写速度可以达到500MB/s，为什么差了这么多呢？

问题就出在我这块移动硬盘是机械硬盘的缘故。机械硬盘的瓶颈在于磁头的读写速度，理论上的极限在60MB/s左右。所以不管我的USB连接线的接口是2.0还是3.0，还是我的mbp的USB接口是2.0还是3.0，读写速度都不会得到太大的提升，可以说几乎没差别。这就是为什么要买SSD固态硬盘的原因，它可以最大限度地释放USB 3.0的传输效率。

所以，不用奇怪为什么买回来的移动硬盘明明支持USB 3.0的传输速率，可是实际却远远不及，问题就在于你的硬盘是机械硬盘。
