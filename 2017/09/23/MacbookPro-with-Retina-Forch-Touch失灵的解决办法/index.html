MacbookPro with Retina Forch Touch失灵的解决办法

September 23 2017

我的MacbookPro with Retina，简称rmbp，型号是Early 2015，每隔一年，触控板就会出现一次问题。上回是2016年，送苹果旗舰店后，直接换了一个触控板，但历经的过程很烦。这一次，电脑莫名卡死后重启，触控板再次失灵。我头痛的是我又要跑一次旗舰店，等上几天时间，费时费神，想想都烦。

无意间，看到一篇叫做Force Touch 觸控式軌跡版竟然失效無法按壓！？先用這兩招自救吧！的文章。抱着侥幸的心态，我想试试看，万一恢复正常了呢。

我先是试了第一种方法，清PRAM，没有成功。抱着最后一丝希望，我又尝试了第二种方法，重置SMC，没想到竟然成功了！

简单介绍一下，为什么会出现Forch Touch失灵的情况？原因是因为Forch Touch触控式轨迹版如果在没有过电的状态下是无法使用的，像我遇到的状况就是不知道为什么机器死机重启后Forch Touch触控式轨迹版却没有正常过电导致无法使用两段式按压。

重置SMC的步骤如下：

  • 将电脑开机
  • 将MagSafe或USB-C电源转换器插入电源与电脑
  • 在内建键盘上同时按下（左侧）Shift-Control-Option键以及电源按钮
  • 同时放开所有按键和电源按钮
  • 按下电源按钮以重启电脑

我的rmbp就是通过重置SMC解决了Forch Touch失灵的问题。还好运气好，只是因为没有正常过电引起的，如果以上两种方法都没用的话，那只能送维修了。
