MacOS访问RHEL(虚拟机镜像)的Nginx服务器

May 23 2017

上一章讲了在RHEL（虚拟机镜像）里搭建完成LNMP框架后，用Firefox打开

http://server_IP/info.php

，验证php安装成功。但是同样的网址用MacOS里的浏览器打开，却看不到任何结果。

解决方案 （1）明确自己的虚拟机使用的网络连接方式，桥接网络还是共享网络，我使用的是共享网络。 （2）确保防火墙是否关闭。在RHEL里查看防火墙状态的代码

systemctl status firewalld

如果状态为active，则需要关闭防火墙。执行

systemctl stop firewalld

。不过，这只是临时的方法，等电脑重启后，防火墙又自动开启了。所以，有必要永久关闭防火墙。执行

systemctl disable firewalld

。如果哪天想要开启防火墙了，执行

systemctl enable firewalld

。在关闭了防火墙之后，我在MacOS里的浏览器能够访问RHEL（虚拟机镜像）里的Nginx服务器了。
