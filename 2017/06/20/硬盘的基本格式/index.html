硬盘的基本格式

June 20 2017

通常来讲，给硬盘分区时，主要考虑三种格式:

一、MS-DOS(FAT)格式

FAT格式的硬盘，无论是Mac还是别的PC，都能对其读写。缺点是单个文件存放不能超过4GB。所以，该格式的硬盘通常被用来存放共享文件，不同操作系统之间都能读写，而且单个文件的限制不超过4GB大小。

二、NTFS格式

NTFS格式的硬盘，在微软操作系统下支持读写，但是在Mac系统下默认是只读的。不过现在Mac上有些软件已经实现了对该格式的硬盘进行读写操作，比如说Mounty。优点是可以存放大文件，所以一般该格式的硬盘用来存放容量大于4GB的文件。

三、Mac OS Extend格式

这种格式的硬盘是Mac系统专有的，只能在Mac上进行读写。一般是用来备份Mac操作系统用的，结合Mac上的Time Machine，除非硬盘坏掉，否则永远也不用担心会丢失Mac上的资料。

总结

以上的三种格式是我们日常生活里最常用到的硬盘格式。怎么给硬盘分区，取决于自己的实际情况。我的Mac的硬盘容量为256GB，我的移动硬盘容量为1TB，我的分区情况

150GB "Shared" FAT 250GB "Time Machine Disk" Mac OS Extend 600GB "Huge Files" NTFS
